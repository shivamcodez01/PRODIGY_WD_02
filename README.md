# PRODIGY_WD_02
Prodigy InfoTech - Task 02: Stopwatch Web App

Task 02 for Prodigy InfoTech Web Development Internship (December 2025)

**Live Demo**  
https://shivamcodez01.github.io/PRODIGY_WD_02/

## Features
- Start / Stop / Reset
- Lap recording (latest laps appear on top)
- Accurate timing using performance.now()
- Responsive, accessible controls
- No external libraries (Vanilla JS)

## Tech
HTML5, CSS3, Vanilla JavaScript

## How to run locally
1. Clone: git clone https://github.com/shivamcodez01/PRODIGY_WD_02.git
2. `cd PRODIGY_WD_02`
3. Open `index.html` in your browser (or run a simple static server).

## Deployment
Publish using GitHub Pages:
1. Push the repo to GitHub.
2. Settings → Pages → Select `main` branch and `/ (root)` (or `master` if applicable) → Save.
3. Visit `https://shivamcodez01.github.io/PRODIGY_WD_02/` after deployment.
